- A application allowing change the desktop background.
- Includes: multihead, xinerama - recall mode that can remember last wallpaper set and cmd line mode for script use.
## Installation
- `sudo pacman -S nitrogen` (always remember to update pacman first).
## Setting wallpaper
- `nitrogen ~/wallpaper` -> if your wallpapers folder is at "/home/username/wallpaper".
- Or open nitrogen in dmenu.
